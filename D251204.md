# Dart 문법 요약

- 작성일 : 2025년 12월 04일
- 범위 : 함수와 제네릭까지

#### 변수
- 값을 담는 상자.값을 재사용하기 위해서 주로 사용함
- 타입 명시
  - `[타입] [변수명] = [담을값];`
  - `int age = 20;`
- 타입 추론 : 타입을 적지 않고 뒤에 오는 값에 따라 자동으로 타입 지정
  - `var [변수명] = [담을값]; // 변경가능 `
  - `final [변수명] = [담을값]; // 변경 불가능`
  - `const [변수명] = [담을값]; // 변경 불가능 => 컴파일 시점에 값이 결정되는 변수`

#### 기본 데이터 타입
- `int` : 정수 
- `double` : 실수(소수점포함)
- `String` : 문자열
- `bool` : 참/거짓
- `dynamic` : 모든걸 담을 수 타입 , 오류 발생 가능성 높아짐


#### 연산자
- 비교 연산자
  - `==` : 같은지 비교
  - `<` : 우측이 큰지 비교
  - `>` : 좌측이 큰지 비교
  - `<=` : 우측이 크거나 같은지 비교
  - `>=` : 좌측이 크거나 같은지 비교
- 산술 연산자
  - `*` : 곱하기
  - `/` : 나누기
  - `%` : 나머지
  - `~/` : 몫

- `a 산술연산자= b` : a 와 b 연산한 결과를 a 에 저장
- `a = b ? c : d` 3항 연산자 : a의 값은 b가 참이면 c 거짓이면 d
- `a = b ?? c` null 병합연산자 : a의 값은 b가 널이면 c 아니면 b

#### 조건문
- `if(조건식) {실행할 코드}`
- `if(조건식1) { } else if(조건식2) {} esle {}`
조건식1 참이 아닐경우 조건식 2, 조건식 2도 참이 아닐경우 else내 중괄호 실행
- `switch` 문도 있음!! => 자동완성 활용하기

#### 반복문
- `for (변수선언; 조건; 증감식) {실행할 코드}`
- `while(조건) {실행할 코드}`

#### 주석
- `command + /` 모든 언어 만능 주석 단축키

#### 컬렉션
- `List<담을 타입> 변수명 = [담을값1,담을값2]` : 여러개의 값을 담을때
- `Set<담을타입> 변수명 = {담을값1,담을값2}` : 중복 허용 X
- `Map(키타입, 값타입) 변수명 = {'키':'값'}` : 사전같은 용도

#### Eunm
- `eunm {멤버1, 멤버2}` 열거형 상수

#### 함수
- `반환타입 함수명 (매개변수) {실행코드}`
- `int add(int a, int b) {return a + b;}` : a와 b를 전달받아서 더한 값을 반환! 사용할 땐 `add(1,2)`
- `int add(int a, int b) => a + b;` : 함수내 return 문만 존재할 경우 화살표로 간단하게 표현 가능
- 반환 타입이 없을경우 `void`
