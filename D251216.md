# Flutter 에서 자주 사용하는 위젯!

- 위젯 : 화면에 보이는 모든 요소

---

1. [***MaterialApp***](https://api.flutter.dev/flutter/material/MaterialApp-class.html)
- 모든 앱에서 반드시 한번은 사용되는 위젯
- 머티리얼 디자인을 사용하는 애플리케이션입니다.
- 머티리얼 디자인 : 구글에서 디자인한 가이드 이름을 머티리얼이라고 지음
- 페이지 설정(앱 내에서 사용되는 페이지들)
- ⭐️ **`Navaigator` 스택 관리**
- **테마 관리**
- 다국어 설정

---

2. [***Scaffold***](https://api.flutter.dev/flutter/material/Scaffold-class.html)
- 페이지 골격을 잡아주는 위젯
- 페이지별로 하나씩
- 많이 나뉘어진 레이아웃을 미리 구글에서 나눠놓은 위젯
- appBar 영역 : 상단, AppBar라는 위젯이 들어감
- body 영역 : 가운데 컨텐츠
- bottomNavigatorBar, bottomSheet : 하단
- 앱을 만들면 MaterialApp이 들어간 위젯 클래스 만들기
- 페이지를 만들 땐 Scaffold가 들어간 위젯 클래스 만들기
- 하나의 클래스는 하나의 파일에서 관리 권장
- 반드시 appBar 영역과 bottom 영역을 사용해야하는건 아님
- 특수한 경우 body안에서 구현하는게 더 편함
(by 박진 2025.12.16)

---

3.[***AppBar***](https://api.flutter.dev/flutter/material/AppBar-class.html)

![AppBar](https://flutter.github.io/assets-for-api-docs/assets/material/app_bar.png)

- Scaffole의 상단영역에 배치할 수 있는 위젯
- 상단역역을 `ledding`, `title`, `actions` 등의 영역으로 미리 골격을 나눠놓은 위젯

- `ledding` : 뒤로가기 버튼 주로 사용
(네비게이션 스택을 보고 이전페이지가 존재하면 자동으로 뒤로가기 버튼을 만들어줘서 안 써줘도 된다.
단, 뒤로가기 아이콘을 바꾸고 싶을땐 구현 해야함)

- ***`title`*** : 제목, 가운데글자.(안드로이드에서는 디폴트로 왼쪽)
`centerTitle` : `true`를 주면 안드로이드에서도 제목이 가운데 정렬됨

- ***`actions`*** : 위젯이 들어가는 리스트.
(리스트 안에 들어가 있는 위젯들이 차례대로 배치됨.)
